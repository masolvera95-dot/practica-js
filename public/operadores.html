Operadores en JavaScript

Existen 2 tipod de operadores:

  • Operadores de asignación
  • Operadores aritmeticos

Operadores de asignación

Abreviaciones Significado
x = y x = y
x += y x = x + y
x -= y x = x - y
x *= y x = x * y
x /= y x = x / y
x %= y x = x % y
x **= y x = x ** y
x <<= y x = x <<= y
x >>= y x = x >> y
x >>>= y x = x >>>= y
AND: x &= y x = x & y
XOR: x ^= y x = x ^ y
x |= y x = x | y

let number = 10;

number += 5;

let frase = "" + numero1 + numero2;
- Da como resultado: 538;

Otro Ejemplo: Uso de funcion "concat();"

let numero1 = "53";
ñet numero2 = 8;
frase = numero1.concat(numero2);
document.write("frase");

Tambien se puede concatenar con "${}".
let nombre = "Alejandro";
let frase = `Soy ${nombre} y estoy aqui.`;
document-write(frase);
- Da como resultado: Soy Alejandro y estoy aqui.
