Condicionales

Los condicionales en JavaScript permiten que el programa tome decisiones según una condición.

Con estructuras como if, else y else if, el código puede ejecutar diferentes acciones dependiendo de si una expresión es verdadera o falsa. Son la base para crear lógica, control de flujo y comportamientos dinámicos en una aplicación.

Ejemplo

let promedio = 9.5;
let ingresosFamiliares = 1200;

if (promedio >= 9.0) {
console.log("Cumples con el promedio mínimo.");

// Este es el IF ANIDADO
if (ingresosFamiliares <= 1500) {
console.log("¡Felicidades! Se te ha otorgado la beca completa.");
} else {
console.log("Promedio excelente, pero tus ingresos superan el límite para la beca.");
}

} else {
console.log("Lo sentimos, el promedio mínimo es de 9.0.");
}
